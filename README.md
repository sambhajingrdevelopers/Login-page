# 3D Water Login Page - Next.js

## Files included
- `app/page.tsx` - Main 3D water login page
- `app/globals.css` - Full styling and animation
- `app/layout.tsx` - Next.js layout
- `Next.tsx` - Standalone TSX copy of the page

## Run
```bash
npm install
npm run dev
```

Open:
```bash
http://localhost:3000
```
